# Name: KeenKeeper:

Keep Your Friendships Alive

# Description:

It's a friends and family tracking platform. This platform keeps track litrally everything. The platform features an intuitive dashboard that categorizes connections into statuses like "On Track" or "Overdue" based on your custom contact goals.Ultimately, it acts as a digital shelf for your social life, providing the tools and reminders necessary to grow a consistent and healthy network.

# Technologies Used:

1. React
2. Tailwind css
3. DaisyUi
4. React router
5. React Recharts, Icon

# Features:

1. Relationship Tracking Dashboard
   Status Overview: At a glance, you can see how many total friends are in your "shelf," how many are "On Track" (recently contacted), and how many "Need Attention" (overdue for a chat).

   Activity Metrics: It tracks "Interactions This Month," likely to help you gamify or measure your social consistency.

2. Smart "Contact Frequency" Alerts
   Each friend card shows a specific time-based status:

   On-Track (Green): You’ve contacted them recently.

   Almost Due (Yellow): It’s getting close to the time you should reach out.

   Overdue (Red): You have exceeded your set goal for contacting that person.

3. Categorization & Tagging
   Context Labels: Friends are tagged with categories like Work, Family, Hobby, and Travel. This helps you keep track of where you know people from and likely allows for filtering your list.
